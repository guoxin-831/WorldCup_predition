评估指标详细解释:
----------------------------------------
准确率 (Accuracy) = 0.5375
  计算公式: Accuracy = (TP + TN) / (TP + TN + FP + FN)
  含义: 预测正确的样本数占总样本数的比例
  解释: 模型在测试集上53.75%的预测是正确的
  范围: [0, 1]，越大越好，1表示全部预测正确
  适用: 各类别样本均衡时最有效

精确率 (Precision) = 0.4669
  计算公式: Precision = TP / (TP + FP)
  含义: 预测为某类的样本中实际为该类的比例(加权平均)
  解释: 模型预测为某类时，46.69%的预测是准确的
  范围: [0, 1]，越大越好
  特点: 衡量预测的准确性，减少误报

召回率 (Recall) = 0.5375
  计算公式: Recall = TP / (TP + FN)
  含义: 实际为某类的样本中被正确预测的比例(加权平均)
  解释: 实际某类的样本中，53.75%被正确预测
  范围: [0, 1]，越大越好
  特点: 衡量预测的全面性，减少漏报

F1分数 (F1-Score) = 0.4583
  计算公式: F1 = 2 * (Precision * Recall) / (Precision + Recall)
  含义: 精确率和召回率的调和平均数
  解释: 综合评价指标，45.83%
  范围: [0, 1]，越大越好，1表示完美分类
  特点: 平衡精确率和召回率，适合类别不均衡场景
