评估指标详细解释:
----------------------------------------
准确率 (Accuracy) = 0.5125
  计算公式: Accuracy = (TP + TN) / (TP + TN + FP + FN)
  含义: 预测正确的样本数占总样本数的比例
  解释: 模型在测试集上51.25%的预测是正确的
  范围: [0, 1]，越大越好，1表示全部预测正确

精确率 (Precision) = 0.5015
  计算公式: Precision = TP / (TP + FP)
  含义: 预测为某类的样本中实际为该类的比例(加权平均)
  解释: 模型预测为某类时，50.15%的预测是准确的
  范围: [0, 1]，越大越好

召回率 (Recall) = 0.5125
  计算公式: Recall = TP / (TP + FN)
  含义: 实际为某类的样本中被正确预测的比例(加权平均)
  解释: 实际某类的样本中，51.25%被正确预测
  范围: [0, 1]，越大越好

F1分数 (F1-Score) = 0.4935
  计算公式: F1 = 2 * (Precision * Recall) / (Precision + Recall)
  含义: 精确率和召回率的调和平均数
  解释: 综合评价指标，49.35%
  范围: [0, 1]，越大越好，1表示完美分类

  结论: 差值较大，模型存在明显过拟合

六、方法局限性
----------------------------------------
1. 历史数据假设: 假设队伍历史表现能够预测未来表现，
   但队伍实力可能随时间变化(如教练更替、球员换代)。
2. 样本量限制: 足球比赛结果受多种因素影响，历史数据有限。
3. 特征局限性: 仅使用历史统计特征，未考虑临场因素。
4. 赛事规则变化: 不同年份世界杯的赛制、规则可能不同。
5. 新队伍问题: 首次参赛的队伍没有历史数据，使用默认值。
